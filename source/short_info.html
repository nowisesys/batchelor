ChemGPS-NP

Instructions for using the ChemGPS-NP web tool:

Upload a correct SMILES-file (the string to the left of eventual IDs) with a maximum of 500 compounds using the browse button and then submit by clicking send file. You can alternatively paste your SMILES in the process data drop box and submit by clicking send data.

To obtain the resulting ChemGPS-NP 8D coordinates go to show job queue in the left menu and download the file by selecting it.
The coordinates can then be plotted using appropriate software.

If you use this application please cite the original publication:

Larsson J, Gottfries J, Muresan S, Backlund A.
ChemGPS-NP: tuned for navigation in biologically relevant chemical space. J Nat Prod. 2007;70(5):789-794.
